    1.1 Introduction to Programming¶

    What is a Program, Programming, and a Programmer?¶

    • Program: A sequence of instructions written to solve a problem.
    • Programming: The process of designing and writing programs.
    • Programmer: A person who solves problems with programming.

    Computer Architecture and Levels of Abstraction¶

    A computer can be viewed as a layered system:

    Physics → Hardware → Assembly Language → Programming Languages → Software

    • Lower layers are low-level and machine-oriented.
    • Upper layers are high-level and human-oriented.

    Programming Languages¶

    A programming language acts as an interface between humans and computers. It must be:

    • readable and expressive for humans,

    • translatable into low-level instructions for machines. High-level programs cannot be executed directly. They must be translated into lower-level code. Two major translation approaches are described below:

    • Interpreter: Executes code line by line (e.g., Python).

    • Compiler: Converts the entire program into machine code before execution (e.g., C/C++). Both approaches bridge the gap between human-written instructions and computer-executable operations.
